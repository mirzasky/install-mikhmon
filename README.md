# install-mikhmon

Install Trmux PlayStore</br>
Munculkan CTRL di Termux : Tekan volume up + q

apt update && apt upgrade -y</br>
apt install curl</br>
curl -o install-mikhmon https://laksa19.github.io/install-mikhmon.txt</br>
chmod +x install-mikhmon</br>
./install-mikhmon</br>

Buka url http://127.0.0.1:8080 di browser Anda.</br>

Command</br>
Start Mikhmon : ./start-mikhmon</br>
Update Mikhmon : ./update-mikhmon</br>
Remove Mikhmon : ./remove-mikhmon</br>
